 🪙 Coin Toss Simulator

Welcome to the **Coin Toss Simulator**, a simple Python mini project where you guess the outcome of a virtual coin toss! 🎯

🚀 Features

- Tosses a virtual coin (Heads or Tails)
- Accepts user guess and compares result
- Uses Python's `random` module
- Clear CLI interaction
- Includes input validation and exception handling

📋 How It Works

1. The program asks the user to start or stop.
2. If "start" is chosen:
    - The user guesses "Heads" or "Tails".
    - The program tosses the coin and compares the result.
    - Displays whether you win or lose.
3. If "stop" is chosen, the program exits.

🛠️ Technologies Used

- Python 🐍
- `random` module
- Basic CLI and input handling

💡 Learning Outcome

- Improved understanding of conditionals
- Working with strings and randomization
- Basic CLI application building
- Handling exceptions (`ValueError`)
